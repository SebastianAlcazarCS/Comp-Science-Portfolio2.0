Vindinium AI
What Is Vinidium?
Vindinium is an multiplayer game in which a player controls an AI (bot) and writes code to control that bot. A player can use any coding language and use it to command his AI in any single way he might want to. The possibilites are endless. The AIs then go into an arena with mines and taverns also including the other players that have qued up their bots. You fight eachother and also the goblins gaurding the mines for control of the map. Control of the map gives you gold, which then gives you a lead in the game because who ever has the most gold at the end of turn 1200
How Does The Bot Work?
The bots works by writing code into a file. The code in this file will make so that your AI will do the commands that you have written into the code. You can write code in any programing language and then run it in the vindinium server and battle your bots to see who has the superior code and strategy.
